Version 3.0.0 is rolling out on Google Play.

Highlights

- 2 basic classes: Knight with a special 1.5× ATK attack, and Paladin with heal.
- Basic translations for Japanese, German, and Spanish.
- Knight special attack reworked.
- Buffed most passives and added a new passive for dodge.
- UI polish: wiggly buttons and a tutorial-like message to help new players.

Notes

- This update includes large code changes to enable translations. Some bugs are likely—please report issues you encounter.
- Want to see your language added? Let us know which one.
- Have ideas for new classes or changes to Knight/Paladin? Share your thoughts!

Upcoming

- Planning to nerf crit damage in one of the next updates.
